# TO-DO Bonito

## System dependencies

- Ruby 3.2.0
- SQLite
- Docker

## Setup

```bash
# Install dependencies
$ dbin/bundle

# Create database and seed data
$ dbin/rails db:setup
```

## Running tests

```bash
$ dbin/rspec
```

## Running linter

```bash
$ dbin/rubocop
```

## Start the application

```bash
$ dbin/rails s
```
